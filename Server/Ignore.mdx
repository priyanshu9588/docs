## How to Setup Ignore Module
Ignore Module is easy to setup just use the commands listed below

## Ignore Management Commmands

- `/ignore add` : Adds a channel to ignore list
- `/ignore delete` : Removes a channel from ignore list
- `/ignore list` : List all the channels in Ignore list

<Note>You Can Always Run `<prefix>ignore` for more information</Note>
