## How to 247 Module
<Info>24/7 is a Premium Module</Info>
247 Module is easy to setup just use the commands listed below
- `/247` : Adds/Removes a channel From/To 247 list

## How to use AutoPlay Module
<Info>AutoPlay is a Premium Module</Info>
Want Groot to never stop playing Music? Use this
- `/autoplay` : Starts/Stops Autoplay Module

## How to change PlayMode
To change playmode just run 
- `/playmode` : Toggles PlayMode from Advance / Basic
